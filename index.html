Financial Complaints

In this project, I created a comprehensive financial complaint dashboard, demonstrating my proficiency in data dashboarding, data cleaning, and exploratory data analysis using Excel and SQL. Spanning from December 2011 to October 2020, the dashboard revealed insights from 75,000 complaints, achieving an impressive 98% timely response rate. "Managing Accounts" emerged as the top complaint category, while "Credit Card" products garnered the highest number of complaints. Additionally, California stood out as the region with the highest complaint volume. With visually engaging data representations, this project exemplifies my ability to transform complex datasets into actionable business insights, making me an asset for any data-driven organization

  • EXCEL and SQL Expertise: Demonstrated proficient data cleaning and exploratory data analysis skills using Excel and SQL, ensuring accurate and meaningful representations of complex datasets.

Car Insurance

  • Developed a dynamic and user-friendly car insurance dashboard using Tableau to analyze claim patterns between male and female policyholders in the automotive industry.
  • Utilized a dataset from Kaggle and carefully cleaned the data in Excel to ensure accuracy and reliability for in-depth analysis.
  • Conducted exploratory data analysis (EDA) with SQL, efficiently answering stakeholder questions and identifying meaningful trends.
  • Key insights revealed that male policyholders had the highest number of claims, with most claims linked to Chevrolet cars, while female policyholders tended to file more claims for accidents involving Ford cars.
  • Effectively communicated the dashboard results to stakeholders through PowerPoint presentations, providing concise and engaging insights with explanatory notes for clarity.
  • The project showcases strong skills in data analysis, data visualization, and storytelling, proving the ability to handle real-world datasets and present actionable findings through visually compelling dashboards and presentations.

  • Data Collection from Websites: I skillfully gathered relevant sales data from various websites, ensuring accuracy and completeness for the analysis.
  • Data Wrangling: Employing advanced data wrangling techniques, I structured and prepared the data for analysis, dealing with missing values and inconsistencies.
  • Data Cleaning with Excel: Using Excel, I efficiently cleaned the dataset, removing duplicates, handling errors, and formatting data to make it suitable for analysis.
  • Data Cleaning with SQL: Leveraging SQL, I further refined the data, performing data transformations, aggregations, and filtering to extract essential insights.
  • Interactive Tableau Dashboard: I designed an interactive and user-friendly Tableau dashboard that presents key performance metrics, trends, and comparisons for Adidas and Nike shoe sales.
  • Visual Representations: To enhance understanding, I used visually compelling charts, graphs, and maps to represent sales patterns, geographical distribution, and product popularity.
  • Performance Analysis: The dashboard provides in-depth analysis of sales performance, highlighting top-selling products, peak sales periods, and revenue trends.
  • Insights and Recommendations: Through data-driven insights, I offered valuable recommendations to optimize sales strategies and capitalize on market opportunities.
  • Portfolio-Ready Report: The final deliverable is a comprehensive report showcasing my proficiency in data analytics, data visualization, and data-driven decision-making.

Mission to Mars

This project showcases my expertise in data analysis and visualization. I collected and analyzed data on successful and unsuccessful attempts to reach Mars using Python and Excel, and then created an interactive dashboard using Tableau. The dashboard provides insights into the history of Mars exploration and highlights the factors that have contributed to the success and failure of these missions.

The dashboard features data on various aspects of each mission, including launch date, launch vehicle, and mission outcome, and allows users to explore the data in a variety of ways. The dashboard is highly interactive and provides a visually appealing representation of the data, making it easy for users to understand and draw insights from the data.

This project demonstrates my ability to work with complex data, use data analysis tools, and create visually appealing dashboards that help organizations make informed decisions. Whether you are a space enthusiast or simply interested in data analysis, this project is sure to captivate and inform.

The concise PowerPoint presentation highlighted key findings, showcasing proficiency in data analysis, visualization, and storytelling.
